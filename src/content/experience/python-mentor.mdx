I mentored two python batches (P15 & P16) with around 100 college students at [http://sheryians.com](http://sheryians.com/) which was one of the most hands on responsibilities i’ve had

most of the students were complete beginners, so the challenge was not just teaching python but to get comfortable with entire computer like installations, vs code, running code, debugging, wrong syntax and general setup issues.

The earlier system followed by other Mentors was to give everyone a sheet of questions and students ask doubts if they ever solve them ( I was a stud there in P13 batch ) So I understood the actual problem while being among them and decided to try something completely new
I was just a high schooler and all stud were in college and other mentors were college grads, so this made me doubt myself for a few days but then I stopped overthinking and started applying my web dev skills to build something

instead of just sharing assignments, I created a simple yet functional platform where :

- studs can solve problem statements ( added by me with example and test cases )
- run their code using a built in editor and compiler
- access daily class files and resources
- see a leaderboard with other batchmates points and stay motivated

the most time consuming part wasn't just teaching concepts but was staying available, resolving repeated issues, and making sure people didn’t drop off when they got struck

this was completely voluenteer work, but it was one of the best experience where I saw a real life impact! after the program, 3 students reached out to me telling they had been placed and that the sessions helped them few such messages made the effort feel meaningful :D
