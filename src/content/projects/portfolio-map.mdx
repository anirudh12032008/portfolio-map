## Overview

Portfolio Map is an experiment in presenting project work as an explorable interface rather than a simple list.
The goal was to make navigation feel intuitive while keeping the structure easy to scan.

## What I built

- A responsive layout that works smoothly across laptop and mobile breakpoints.
- Structured data-driven sections so adding or updating items stays simple.
- Lightweight transitions to make state changes feel clear without being distracting.

## Lessons

The biggest win was keeping the architecture simple from the start.
Small, predictable components made it easier to iterate quickly on both visuals and content.
